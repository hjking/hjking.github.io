PC端工具

系统相关

  • Process Explorer Windows 上任务管理器的升级版。每个进程的子进程和母进程是什么都一目了然。

  • Dism++ Windows实用系统优化工具

  • GeekUninstaller 卸载软件，最大的好处是能直接跳转到注册表。

  • cmder windows下cmd的替换工具,支持PowerShell。

  • Chocolatey 软件包管理系统，完全用命令行来安装应用程序。

  • ChocolateyGUI Chocolatey GUI 界面

其他

  • Everything 管理搜索文件的工具，简直神速，支持正则表达式，输入关键字实时显示搜索结果，秒杀Windows自带的搜索。

  • SpaceSniffer 磁盘分析软件，它会用方块画出我们磁盘文件的占用情况

  • Flux Flux 晚上会根据时间把你的屏幕调黄，有改善睡眠的作用。

  • Total Commander 替代自带的资源管理器，功能更强大。

  • rufus 制作启动U盘的利器。

  • Bandizip 压缩软件

  • Recuva 数据恢复软件

  • Ditto 剪切板记录管理

  • Dexpot 虚拟桌面

  • SnapX 自定义窗口排布方式和大小

快捷启动

  • Keypirinha 应用启动器

  • Quicker 为常用操作建立捷径，让效率触手可及！

  • executor 通过关键字快速启动程序，打开网页等。

图像视频

  • FastStone Capture 截图工具

  • Snipaste 截图工具

  • etcher Burn images to SD cards & USB drives, safely and easily.

  • 百度云满速下载 使用本地http代理服务器方式嗅探下载请求，支持所有操作系统和大部分主流浏览器,支持分段下载和断点下载。

Markdown

  • typora

    Typora 将「编辑写字」、「预览呈现」两个过程合并了，所有行内元素（比如粗体、斜体）会根据是否在编辑状态而自动在编辑和预览状态之间切换，区块元素（比如标题、列表）会在按下 Enter 后及时渲染。

  • Boostnote

    一个专门为程序员设计的笔记应用：支持上百种程序语言的高亮显示、项目 Dev Track 进度追踪、独立的 Finder Popup显示窗口，用户在新建环节甚至可以创建多个 Tab，用来组成独立的代码项目。目前，Boostnote 支持 Windows、macOS、Linux 三大桌面端和 Android、iOS 两大移动端平台。

  • Laverna

    Dropbox 同步; Remote Storage 存储协议; 加密功能。

  • Simplenote

    支持平台最全面的写作工具，支持包括 Windows、macOS、Linux、Android、iOS 和 Web 共六大平台，覆盖桌面端和移动端，并且所有文档都可以实现跨平台同步编辑。
